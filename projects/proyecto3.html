Proyecto 3: Análisis de rendimiento comercial (Caso Amazon)

Estudio analítico de la evolución de ventas, comportamiento por categorías de productos y participación de mercado

Volver al Portafolio

Detalles del Proyecto

Descripción

Se busca realizar un análisis detallado que permita identificar tendencias de consumo, productos líderes y variaciones estacionales en las ventas. El proyecto combina limpieza, procesamiento y visualización de datos para generar insights claros y accionables, utilizando dashboards interactivos, gráficos comparativos y reportes estadísticos. El objetivo es comprender el desempeño comercial de la empresa y facilitar la toma de decisiones estratégicas sobre productos, categorías y mercados.

Herramientas utilizadas

  • Power BI: Para Crear dashboards interactivos y visualizaciones dinámicas de ventas, categorías y participación de mercado
  • Python: Para tendencias, crecimiento porcentual y comparaciones entre categorías
  • RStudio: Para regresiones o segmentación de productos por comportamiento de ventas
  • Datasets públicos: Para comparar periodos, identificar tendencias y validar hallazgos

Fuentes de Datos

Reportes clave: Power BI · Python · datasets públicos · R Studio
