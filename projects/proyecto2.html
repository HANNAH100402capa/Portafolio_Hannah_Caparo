Detalles del Proyecto

Descripción

Se busca realizar un análisis comparativo de exportaciones, importaciones y participación de cada país en el comercio mundial, con un enfoque en sectores clave como tecnología, energías renovables, manufactura avanzada y e-commerce. El proyecto incluye la limpieza, el procesamiento y la visualización de datos mediante gráficos, mapas y dashboards, con el objetivo de identificar tendencias globales, oportunidades de inversión y patrones de desarrollo económico. Este análisis permite comprender mejor la dinámica del comercio internacional y facilita la toma de decisiones estratégicas para empresas, inversores y responsables de políticas públicas.

Herramientas que se utilizarán

  • Excel: Para explorar, limpiar y organizar datasets pequeños o medianos
  • python (pandas, numpy): Para limpieza avanzada, procesamiento y análisis de grandes volúmenes de datos
  • RStudio: Para análisis profundo y visualizaciones, exploración y estudio de tendencias.
  • Power BI: Para visualizar y comunicar los datos de manera clara e interactiva

Fuentes de Datos

Reportes clave: World Bank Data · WTO Trade Statistics · UN Comtrade Database · IMF Data
